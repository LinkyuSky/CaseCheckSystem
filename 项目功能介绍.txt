================================================================================
                        检察机关案卡冲突自动审核系统
                            功能与特色介绍
================================================================================

一、项目定位
--------------------------------------------------------------------------------
本系统是基于规则引擎的智能案件数据质量检测平台，专为检察机关审查逮捕和
审查起诉业务设计，实现案卡数据的自动化审核，提升数据准确性和工作效率。

二、核心功能模块
--------------------------------------------------------------------------------

【模块一：智能规则引擎】
├─ 模块化架构设计：新增规则只需添加函数，无需修改主程序
├─ 规则库覆盖：24+条专业审核规则（R001-R024）
├─ 跨表关联检测：支持审查逮捕表(010101)与一审公诉表(020101)联动校验
└─ 智能冲突识别：自动检测案由矛盾、日期异常、字段缺失等问题

【模块二：设备身份认证系统】
├─ 机器码生成：基于UUID硬件标识，16位唯一编码
├─ MAC地址提取：自动获取网络接口物理地址
├─ IP地址识别：UDP连接探测获取内网IP
└─ 设备绑定授权：激活码与硬件信息强绑定，防止盗版

【模块三：激活码生成与验证机制】
├─ 加密算法：SHA256哈希 + XOR加密
├─ 激活码格式：10位数字编码
├─ 过期检测：到期前3天预警提醒
└─ 防重用机制：记录已使用激活码，防止重复激活

【模块四：智能日期校验】
├─ 系统日期检测：防止回滚时间绕过授权
├─ 审结日期合规性：审查逮捕案件7天时限校验
└─ 执行情况预警：审结后3天未填录自动提醒

│ R007    │ 审结后长期未填录执行情况预警                                      │
│ R008    │ 侵犯人身权利案件被害人情况校验                                    │
│ R009    │ 故意伤害等罪名案件造成伤亡情况检测                                │
│ R010    │ 掩饰隐瞒犯罪所得具体情形漏填检测                                  │

四、内置罪名库分类
--------------------------------------------------------------------------------
├─ 职务犯罪：贪污罪、受贿罪、挪用公款罪等40+罪名
├─ 危害国家安全犯罪：背叛国家罪、间谍罪等20+罪名
├─ 经济犯罪：生产销售伪劣商品罪、金融诈骗罪等100+罪名
├─ 侵犯公民人身权利犯罪：故意杀人罪、强奸罪等50+罪名
└─ 伤亡类犯罪：故意伤害罪、交通肇事罪等10+罪名

五、技术架构特色
--------------------------------------------------------------------------------
├─ 模块化设计：规则独立封装，易于扩展和维护
├─ 数据安全：本地处理，数据不上传，保护办案机密
├─ 多格式兼容：支持YYYY-MM-DD和YYYY/MM/DD日期格式
├─ 字段智能清洗：自动处理空值、格式转换
└─ 跨表数据关联：通过嫌疑人编号追踪案件流转

六、部署方式
--------------------------------------------------------------------------------
├─ 便携版：单EXE文件，无需安装，绿色运行
├─ 授权方式：设备识别码+激活码机制
├─ 运行环境：Windows系统，支持离线使用
└─ 输出格式：Excel格式冲突检测报告

七、典型应用场景
--------------------------------------------------------------------------------
1. 审查逮捕案件审核
   ├─ 自动检测移送罪名与涉嫌罪名的一致性
   └─ 审查批准逮捕案件的前科记录完整性

2. 一审公诉案件审核
   ├─ 追踪纠正漏捕嫌疑人的后续处理情况
   └─ 验证案件造成伤亡情况与罪名的匹配性

3. 数据质量监控
   ├─ 定期批量检测，输出质量报告
   └─ 重点关注涉医、涉外、校园暴力等特殊案件

八、创新亮点
--------------------------------------------------------------------------------
├─ 零代码规则扩展：新增规则只需添加check_Rxxx函数
├─ 智能字段映射：自动适配不同表结构的字段差异
├─ 自适应日期格式：智能识别多种日期格式
├─ 跨表联动检测：实现案件全流程数据一致性校验
└─ 设备强绑定：三重硬件信息绑定，保障软件授权安全
